Bienvenido a MADGA

Aún no hay un Site configurado. Crea uno para empezar a publicar contenido.

  1. 1. Crea un superuser:
    python manage.py createsuperuser
  2. 2. Crea tu primer Site:
    python manage.py madga_create_site
  3. 3. Entra a /studio/ con tu superuser y empieza a publicar.
