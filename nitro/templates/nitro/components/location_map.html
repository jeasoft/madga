{% load i18n l10n %} {% comment %} Reusable location picker — MapLibre map with click-to-place draggable marker. Required context vars: map_id — unique DOM id for the map container (e.g. "str-prop-map") lat_field — name of the hidden latitude input (e.g. "latitude") lng_field — name of the hidden longitude input (e.g. "longitude") Optional context vars: initial_lat — pre-filled latitude (default: Santo Domingo center) initial_lng — pre-filled longitude initial_location_id — UUID of the AdministrativeArea currently selected on the property (provincia/municipio/sección/sector). If no point is set, we'll center on this area. zoom — initial zoom level (default: 12) height — Tailwind height class (default: "h-72") cascade_inputs — comma-separated list of input names to watch for cascade changes (default: "province,municipality,section,sector"). When any of these change, the map flies to the picked area's centroid. Works inside HTMX-injected slidelovers because the component data is defined inline in x-data (no Alpine.data() registration race). {% endcomment %} {% localize off %}
